Windows 95: Click on the graphic to preview the wallpaper. Once the entire image is displaying, click on the right mouse button and select "Save as Wallpaper."

Windows 3.1 and Windows for Workgroups:
1. Click on the graphic to preview the wallpaper. Once the entire image is displaying, click on the right mouse button and select "Save as..."

2. Once the file has downloaded to your hard drive, copy it to the \WINDOWS\ directory, usually C:\WINDOWS\.

3. Open the Control Panel program, usually located within the Main Program group.

4. Open the Desktop applet.

5. In the Wallpaper section (located towards the lower left), select the file from the drop down menu.

6.Click the Tile radio button.

7.Click the OK button. That's it!
